Image Search Abstraction API

Description

1. This API returns in JSON format the image URLs, alt text and page URLs for a set of images relating to a given search string.

2. You can paginate through the responses by adding ?offset=2 parameter to the URL.

3. You can get a list of the ten most recently submitted search strings.

  • Instructions

  • Perform a search with the path https://all-the-images.herokuapp.com/search/<query>, replacing <query> with your search string.
    Adding ?offset=<page-number> will paginate the results, 1 being the default, 2 being the second page, etcetera.

  • View the past ten search terms by visiting https://all-the-images.herokuapp.com/history
